Selected Work

APOD 2.0

  • React.js
  • Firebase
  • REST API

APOD 2.0 is a project designed to update NASA's Astronomy Picture of the Day website for the modern web.

Launched in 1995, the NASA'S APOD serves a new beautiful astronomy picture every single day. Unfortunately the interface has not been updated for the last 21 years.

Using NASA's public API I have modernized APOD with a filmstrip scroll, keyboard navigation, asynchronous image preloading, and more. The result is an improved, smooth, and enjoyable APOD picture viewing experience.

Tectonic Newsletter

  • React.js
  • Firebase
  • Bootstrap
  • Zurb Foundation

I helped to roll out Tectonic's brand new weekly design newsletter.

This project had many constraints to work around, chiefly the limitations of HTML email design and the need to automate newsletter formatting and image processing.

Using Zurb Foundation to create a responsive HTML template, I created a tool that can easily be used by anyone to fill in and process newsletter content.
