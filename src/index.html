Selected Work

APOD 2.0

  • React.js
  • Firebase
  • REST API

APOD 2.0 is a personal project designed to update NASA's Astronomy Picture of the Day website for the modern web.

Launched in 1995, apod.nasa.gov serves a new beautiful astronomy picture every day. Unfortunately the interface has not been updated for the last 21 years.

Using NASA's public API I have modernized APOD with a scrolling filmstrip, keyboard navigation, asynchronous image preloading, and more. This project is ongoing, open source, and available on github.

UW Design 2015

  • jQuery
  • Isotope Masonry
  • REST API

This website showcases the student profiles and work of the University of Washington Department of Design's 2015 graduating class.

Of the eight students assigned to the project, I was the primary person responsible for coding the site from scratch. I used the tumblr microblogging platform to make it easier for participating students to submit their work.

Tectonic Newsletter

  • React.js
  • Firebase
  • Bootstrap
  • Zurb Foundation

While at Tectonic I helped to roll out a brand new weekly newsletter.

The first part of the project involved designing the newsletter and accounting for modern HTML email constraints. A template was built using Zurb Foundation to maximize responsiveness and consistency.

I also built an interactive form that enables users to fill in the template with arbitrary content and export the result. This greatly streamlined the weekly production of newsletter campaigns.
